== Description ==
This adds custom functionality to create full screen width slides and use them anyhere using shortcode generated by plugin.

PS: This is custom made for enfold theme and supports Avia builder for making slides.

== Changelog ==

* Version 1.0 - Initial release.

== Installation ==

There are 2 ways of installation. If your setup supports it, you can search for the plugin, by name, within your WordPress control panel and install it from there.
Alternatively, you download the .zip file, unzip it, and upload folder to the `wp-content/plugins/` folder of your WordPress instaltion folder.
